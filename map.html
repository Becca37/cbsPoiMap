This map contains places we've been and places we might someday want to go. It's mainly meant so that we can look up places when we're out and about, but others might find some interesting places to visit too. Happy browsing. :0)
Map Controls
  • To toggle the traffic layer, click the traffic control.
  • To toggle the clustering of markers, click the clustering control.
  • To toggle troubleshooting, click the test control.
  • To show your current location click the locator control (bottom left area). You will need to allow the browser to use your location. This will then:
    • Display the GPS coordinates and accuracy of your location in the locator control area.
    • Start watching your movements (less stalker-y than it sounds: location data is not stored).
    • Display your approximate current location on the map with a pulsating blue circle:
    Once location tracking has been started you can:
    • Click the crosshairs again at any time to pan/zoom to your current location.
    • Click the current location icon on the map to show additional information in this side panel about, yup, you guessed it, your current location.
To view details of any marker click its icon. Information for any normal Google Maps icons will show data in a popup window, with a link to the standard Google Maps. The information for any icons added by Chasing Blue Sky will display in this side panel ...
  • Visited If we've personally visited the location, the eyeball icon will be green. If we haven't, it will be gray with a slash through it ().
  • Website If we've recorded a reference website for the location, the globe icon will be blue and clickable.
  • Navigator The navigator icon will be blue and clickable. Click it to start navigating in Google Maps.
  • If a marker we've added interests you, you can either:
    • Grab the GPS coordinates for use in your preferred mapping program. e.g. --> Coordinates 39.5261176, -105.0265309
    • Add the location as a stop in a Furkot trip. The second icon there shows the icon which will be used by default within the Furkot interface.

      To state the obvious, we think using Furkot for trip planning is a great option, and we use it ourselves, or we would not have integrated it. Visit Furkot Site :0)
Compatibility / Requirements This does require an internet connection. Use it on a device with GPS if you're concerned about getting the best location accuracy. It should work on current browsers, both desktop and mobile. However, we make no promises about it being pretty or easy to work with on small screens*, on any given device, or on any given operation system. We have used it ourselves on Chrome (desktop, android Galaxy Note 9, iOS iPad Pro), Firefox (desktop), and Safari (iPad iPad Pro).

* If you are on a small device such as a smartphone, landscape mode and pinch-and-zoom is your friend. That or a magnifying glass.
